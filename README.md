# biliup
因为实习在公司做的都是爬虫的二次开发（单纯开发spider），很想尝试一下自己开发一个之后优化。另外一个，秋招来临，感觉项目有点不对口，算是补充一下。  主要的过程，先把基本的爬虫写出来，写好MySQL存储。之后添加IPProxy代理池，之后加入Scrapy-Redis，将阿里云作为master。  目前做到这边，后续研究下MySQL连接池，然后打算用Docker部署，数据爬取之后就是数据分析了。
